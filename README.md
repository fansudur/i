# ∅ | UN

A mirror-reflection installation.
镜面反射装置。

[fansudur.github.io/i](https://fansudur.github.io/i)

---

A sphere — Σ — is the origin of sight, where light begins. It looks out through a single aperture, F. Beyond F stands a chain of mirrors, each tilted 45°. With every reflection the path of light turns 90°, and the world unfolds again, one scale larger than the last.

Red, green, and blue are not three separate worlds. They are three layers of one world — three depths of the same seeing. They share the same Σ and the same F, and at the vantage of Σ they resolve into a single, complete image.

You stand where Σ stands. Through F, you see the whole.

球体 Σ 是视线的起点，光从这里开始。它透过唯一的取景框 F 望出去。F 之外，是一列 45° 倾斜的镜面；光每反射一次，光路便旋转 90°，世界随之再次展开，比上一层更大一级。

红、绿、蓝不是三个独立的世界。它们是同一个世界的三个层级 —— 同一次观看的三重深度。它们共享同一个 Σ、同一个 F，在 Σ 的视点处合成为一幅完整的画。

你站在 Σ 的位置。透过 F，你看见这幅画的全貌。

## Run · 运行

Zero build. Pure static. Works offline.
零构建，纯静态，离线可用。

```bash
# Serve this directory as the static root · 把本目录当静态根
python -m http.server 8765
# → http://localhost:8765
```

Any static server will do; the work uses no browser storage. Or simply visit the live work:
任意静态服务器皆可，本作不使用任何浏览器存储。或直接访问在线版：

**[fansudur.github.io/i](https://fansudur.github.io/i)**
