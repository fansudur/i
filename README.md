# ∅ | UN — Incidence

Incidence is a mirror-reflection installation by FANSUDUR.

Σ is the origin of sight. From Σ, a single aperture opens into three optical scenes. The reflected path turns through a chain of mirrors, and the visible world folds into one image from the viewing position.
